## System Requirements

- Linux (Arch, Ubuntu, Debian, Fedora, PopOS)
- NVIDIA GPU with CUDA support
- NVIDIA Driver (Open or Proprietary)
- Container runtime (Docker, Podman, or Containerd)
- Rust 1.70+ (for building from source)
